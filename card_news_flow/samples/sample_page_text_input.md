# 샘플 자연어 입력

아래처럼 한 번에 작성하면 LLM 입력 정리 노드가 카드뉴스 ver2 표준 JSON으로 정리합니다.
각 페이지는 `제목 / 소제목 / 본문 / 이미지 / 하이퍼링크`를 넣을 수 있고, 필요 없는 항목은 생략해도 됩니다.

하이퍼링크 작성법:
- `하이퍼링크: 화면에 보일 글자 | https://example.com/path`
- 화면에는 `화면에 보일 글자`만 보이고, 클릭하면 뒤의 실제 URL로 이동합니다.
- URL은 `http://` 또는 `https://`로 시작해야 합니다.

본문: 팀 안에서 활용 기준을 함께 맞추면 결과 품질도 더 안정적으로 관리할 수 있습니다.

[3페이지]
제목: 바로 써볼 수 있는 프롬프트 팁

이미지: prompt_tip
본문: "누가 읽는지", "무엇을 얻고 싶은지", "어떤 형식이 필요한지"를 함께 쓰면 답변이 더 명확해집니다.
하이퍼링크: 프롬프트 예시 보기 | https://example.com/prompt-tips

[4페이지]

소제목: 입력해도 되는 정보와 안 되는 정보를 먼저 구분합니다.
이미지: security_check
본문: 사내 기밀, 개인정보, 고객 정보는 외부 AI 도구에 입력하지 않습니다.
본문: 공유 전에는 민감 정보가 포함되어 있지 않은지 한 번 더 확인해 주세요.

이미지: action_check
본문: 내 업무에서 반복적으로 작성하거나 정리하는 일을 하나 골라 AI를 적용해 보세요.
