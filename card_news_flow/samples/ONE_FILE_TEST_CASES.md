# 카드뉴스 생성 Flow 샘플 입력

## 1. 사내 AI 활용 문화 확산

```text
이번 달 사내 카드뉴스 주제는 AI 활용 문화 확산입니다.
주요 내용은 업무 자동화 사례 공유, 보안 유의사항, 프롬프트 작성 팁, 다음 달 교육 안내입니다.
톤은 밝지만 너무 가볍지 않게, 직원들이 바로 읽고 행동할 수 있게 만들어주세요.
마지막 카드에는 교육 신청 버튼을 넣고 싶습니다.
```

확인 포인트:

- 6장 내외 카드뉴스 생성
- 16:9 가로형 SNS 카드뉴스로 렌더링
- 아래로 스크롤되지 않고 카드 클릭/버튼으로 화면 전환
- 보안 유의사항 카드 포함
- 마지막 카드 CTA 포함
- 캐릭터가 cover와 closing 카드에 반복 사용되고 idle animation으로 살짝 움직임

## 2. 품질 개선 캠페인

```text
다음 달 품질 개선 캠페인을 안내하는 카드뉴스가 필요합니다.
핵심 메시지는 불량을 숨기지 않고 빠르게 공유하면 개선 속도가 빨라진다는 것입니다.
내용에는 조기 공유의 장점, 보고 채널, 좋은 보고 예시, 참여 보상을 넣어주세요.
버튼은 품질 개선 제안서 작성 페이지로 이동하게 만들고 싶습니다.
```

확인 포인트:

- 캠페인 톤 유지
- 제안서 작성 CTA 버튼 포함
- 단계별 실천 방법 카드 포함

## 3. 보안 수칙 리마인드

```text
월간 보안 수칙 리마인드 카드뉴스를 만들어주세요.
주제는 외부 AI 도구 사용 시 회사 기밀과 개인정보를 입력하지 않는 것입니다.
직원들이 겁먹기보다는 구체적으로 무엇을 하면 안 되는지 이해했으면 좋겠습니다.
마지막에는 보안 문의 채널로 이동하는 버튼을 넣어주세요.
```

확인 포인트:

- 금지 행동과 권장 행동이 분리됨
- 과도한 공포 조성 표현 회피
- CTA URL이 없으면 버튼은 보안 문의 안내 anchor로 fallback

## 4. 특정 페이지 이미지 그대로 사용

카드뉴스 내용:

```text
이번 달 AI 뉴스레터는 AI 활용 우수 사례 공유입니다.
총 6개 화면으로 구성하고, 2026년 7월호 발간호 정보를 첫 화면에 표시해주세요.
3페이지에는 이미 디자인팀이 만든 이미지를 그대로 넣을 예정이므로 별도 문구를 만들지 말아주세요.
나머지 페이지는 사례 소개, 실천 팁, 보안 유의사항, 교육 신청 CTA로 구성해주세요.
```

`페이지 이미지 대체 JSON`:

```json
[
  {
    "page": 3,
    "data_uri": "data:image/png;base64,PUT_APPROVED_BASE64_HERE",
    "alt": "AI 활용 우수 사례 소개 이미지",
    "fit": "contain",
    "background_color": "#FFFDF7"
  }
]
```

확인 포인트:

- 6개 화면 순서 자체는 고정 유지
- 3페이지는 headline/body/bullets/buttons/character 없이 이미지 전용 화면으로 렌더링
- 3페이지 카드 영역 클릭 시 4페이지로 이동
- 이미지 `fit`을 `cover`로 바꾸면 카드 영역을 채우는 방식으로 표시
